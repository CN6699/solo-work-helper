# 心智障碍者职场适配全流程支持助手项目总结

## 项目概述
基于TRAE SOLO打造的零代码、零安装、现场即用、适配心智障碍者认知特点的轻量化职场支持工具，面向心智障碍就业群体、一线就业辅导员、爱心企业带教老师三大核心用户。

## 核心功能模块
1. **岗位工作流程可视化拆解工具**：解决心智障碍者多步骤工作记不住、做不对的核心痛点，生成适配认知特点的分步操作指引。
2. **职场沟通辅助与话术生成工具**：解决心智障碍者职场不会说、不敢说、说不清的痛点，降低人际沟通焦虑。
3. **情绪与需求表达支持工具**：解决心智障碍者无法准确识别、表达自身情绪和需求的痛点，减少情绪应激事件。
4. **岗位适配训练与岗前培训工具**：解决就业辅导资源紧缺、培训材料制作效率低的痛点，降低企业用工门槛。

## 技术实现
1. **基于TRAE SOLO**：100%使用TRAE SOLO实现，无需编程基础，零代码操作。
2. **系统提示词固化**：设置核心角色设定、服务对象、必须遵守的核心规则，确保输出内容符合心智障碍者认知特点。
3. **专项Prompt开发**：针对四大功能模块分别开发专项Prompt，确保生成内容的专业性和适配性。
4. **多轮对话优化**：实现分步引导式对话，用户无需一次性输入所有参数，降低使用门槛。
