# 心智障碍者职场适配支持助手使用示例

## 系统提示词使用方法
1. 打开TRAE SOLO
2. 复制 `/workspace/prompts/system_prompt.md` 中的系统提示词到SOLO中
3. 开启长期对话记忆功能
4. 开始使用专项Prompt

## 模块1：岗位工作流程可视化拆解工具
**示例输入：**
```
基于上述核心角色设定，为【岗位名称：超市理货员】的【具体任务：临期商品下架】，生成适配心智障碍者的操作指引，严格遵守以下规则：
1.  步骤拆分：将完整任务拆分为不超过8个核心步骤，单步骤只包含2个以内的动作，用短句表述，通俗易懂；
2.  固定输出结构：
- 任务核心目标（一句话说清要做什么、做到什么标准）
- 分步操作指引（编号+动作+完成标准，无复杂表述）
- 易错点提醒（不超过3条，每条一句话）
- 做错了怎么办（3步以内的应急处理方案，优先找带教老师）
3.  所有内容可直接打印成口袋卡，心智障碍者可照着步骤独立完成，无需二次修改。
```

**示例输出：**
- 任务核心目标：把快过期的商品从货架上拿下来，交给带教老师
- 分步操作指引：
  1. 找到临期商品（看商品上的日期）
  2. 轻轻取下商品，放在手上
  3. 把商品拿到带教老师面前
  4. 告诉带教老师：这是临期商品
- 易错点提醒：
  1. 别拿错保质期好的商品
  2. 取商品时别弄掉其他商品
  3. 记得找带教老师，别自己处理
- 做错了怎么办：
  1. 立刻停下手中的事
  2. 找到带教老师
  3. 告诉带教老师哪里做错了

## 模块2：职场沟通辅助与话术生成工具
**示例输入：**
```
基于上述核心角色设定，为【岗位名称：烘焙店前台帮工】生成职场沟通话术，严格遵守以下规则：
1.  覆盖4个核心场景：顾客常规提问、同事工作对接、向带教老师求助、表达自身需求；
2.  每个场景生成不超过5条高频话术，单条话术不超过20个字，用短句、大白话，心智障碍者可直接背诵、直接说出口；
3.  额外生成3条突发场景应急话术（如被顾客投诉、做错事被批评），话术必须简单、无攻击性，优先引导找带教老师；
4.  所有话术必须贴合岗位真实场景，可直接在工作中使用。
```

**示例输出：**
- 顾客常规提问话术：
  1. 您好，需要什么帮助？
  2. 这个面包是今天做的，新鲜。
  3. 收款机在这里，麻烦扫码。
  4. 谢谢光临，欢迎再来！
  5. 请问需要打包吗？
- 同事工作对接话术：
  1. 这个订单我做完了，给你。
  2. 请问这个放在哪里？
  3. 我去洗手间，帮我看一下。
  4. 这个面包卖完了，告诉顾客。
  5. 下班时间到了，交接一下。
- 向带教老师求助话术：
  1. 老师，我不会做这个，能教我吗？
  2. 老师，这个机器怎么用？
  3. 老师，顾客问的问题我不会答。
  4. 老师，我身体有点不舒服。
  5. 老师，我需要休息一下。
- 表达自身需求话术：
  1. 我想喝水，去拿杯子。
  2. 我肚子饿了，要去吃饭。
  3. 我眼睛累了，想休息5分钟。
  4. 我手疼，需要戴手套。
  5. 我要上洗手间，马上回来。
- 突发场景应急话术：
  1. 对不起，我去找带教老师。
  2. 老师，这里有顾客投诉我。
  3. 老师，我做错了，能帮我吗？

## 模块3：情绪与需求表达支持工具
**示例输入：**
```
基于上述核心角色设定，生成情绪与需求表达支持内容，严格遵守以下规则：
1.  情绪识别与分级指引：用可视化的方式，帮助心智障碍者识别自己的情绪状态（平静/紧张/焦虑/生气），并对应生成极简的情绪舒缓步骤（如深呼吸3次、找带教老师、到休息区坐5分钟）；
2.  需求表达模板：生成「我需要帮助」「我身体不舒服」「我想休息5分钟」等标准化需求表达模板，配简单图示，心智障碍者可直接使用，无需组织语言；
3.  为带教老师生成「情绪应激事件处理指引」，提前预判风险，给出标准化的应对方法，避免事态升级；
4.  所有内容必须用短句、大白话，单句不超过15个字，可直接在工作现场使用。
```

**示例输出：**
- 情绪识别与分级指引：
  1. 平静：心情好，工作顺利
     - 继续保持，认真工作
  2. 紧张：心跳加快，手心出汗
     - 停下来，深呼吸3次
     - 喝口水，休息1分钟
  3. 焦虑：担心做不好，害怕被批评
     - 找到带教老师
     - 告诉老师：我有点担心
  4. 生气：心里难受，想发脾气
     - 离开现场，到休息区
     - 坐5分钟，慢慢平静
- 需求表达模板：
  1. 我需要帮助：老师，我不会做这个
  2. 我身体不舒服：老师，我肚子有点疼
  3. 我想休息5分钟：老师，我有点累
  4. 我渴了：老师，我想喝水
  5. 我饿了：老师，我想吃饭
- 情绪应激事件处理指引（给带教老师）：
  1. 保持冷静，不要大声说话
  2. 走到心智障碍者身边，轻声询问
  3. 用简单的话安抚：「没关系，我来帮你」
  4. 带他到安静的地方，慢慢沟通
  5. 了解原因，给予具体帮助

## 模块4：岗位适配训练与岗前培训工具
**示例输入：**
```
基于上述核心角色设定，为【岗位名称：酒店保洁】生成岗位适配训练与岗前培训内容，严格遵守以下规则：
1.  一键生成岗位完整培训手册：包含岗位认知、工作流程、沟通规范、安全规则、情绪管理五大模块，可直接使用，无需二次修改；
2.  岗位模拟训练题库：生成岗位核心技能的模拟训练题、实操考核标准，可直接用于岗前培训和日常考核；
3.  企业用工适配指引：为爱心企业生成「心智障碍员工用工管理指引」，包含带教老师工作规范、环境适配建议、激励方法、风险防控等内容；
4.  所有内容必须用短句、大白话，单句不超过15个字，适配心智障碍者的认知特点，可直接在工作现场使用。
```

**示例输出：**
- 岗位完整培训手册：
  1. 岗位认知：
     - 酒店保洁的工作是保持客房干净整洁
     - 要对顾客友好，保持微笑
  2. 工作流程：
     - 敲门进入客房
     - 整理床铺，换床单被罩
     - 清洁卫生间，擦镜子
     - 吸尘，拖地
     - 检查物品，补充消耗品
  3. 沟通规范：
     - 遇到顾客说：您好！
     - 顾客问话要认真回答
     - 有问题找带教老师
  4. 安全规则：
     - 小心使用清洁剂，别弄到眼睛里
     - 拖地后要放防滑牌
     - 发现危险要告诉带教老师
  5. 情绪管理：
     - 工作累了可以休息一下
     - 心情不好找带教老师聊聊
- 岗位模拟训练题库：
  1. 如何正确整理床铺？
     - 答案：先把旧床单撤下来，再铺新床单
  2. 遇到顾客询问房间设施，应该怎么做？
     - 答案：微笑回答，不知道的找带教老师
  3. 清洁卫生间时要注意什么？
     - 答案：用对清洁剂，注意安全
- 企业用工适配指引（给爱心企业）：
  1. 带教老师工作规范：
     - 耐心教导，多鼓励
     - 分解任务，一步一步教
  2. 环境适配建议：
     - 工作区域保持整洁有序
     - 物品摆放固定位置
  3. 激励方法：
     - 完成任务给予表扬
     - 定期评选优秀员工
  4. 风险防控：
     - 定期安全培训
     - 建立应急处理机制
