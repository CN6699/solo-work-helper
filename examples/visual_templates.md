# 心智障碍者职场适配支持助手可视化模板

## 1. 岗位工作流程卡模板

### 设计原则
- 使用大字体，清晰易读
- 每个步骤配有简单图示
- 使用颜色区分不同步骤
- 保持页面简洁，避免干扰元素

### 模板示例
```
┌────────────────────────────────────────┐
│              工作流程卡                │
│        岗位：超市理货员                │
│        任务：临期商品下架              │
├────────────────────────────────────────┤
│                                        │
│  1. 🔍 找到临期商品                   │
│     看商品上的日期                      │
│                                        │
│  2. 📦 轻轻取下商品                   │
│     放在手上                           │
│                                        │
│  3. 👨‍🏫 拿到带教老师面前                │
│                                        │
│  4. 💬 告诉带教老师                     │
│     这是临期商品                        │
│                                        │
├────────────────────────────────────────┤
│  ⚠️ 易错点：                          │
│  1. 别拿错保质期好的商品                │
│  2. 取商品时别弄掉其他商品              │
│  3. 记得找带教老师，别自己处理          │
├────────────────────────────────────────┤
│  ❌ 做错了怎么办：                     │
│  1. 立刻停下手中的事                    │
│  2. 找到带教老师                        │
│  3. 告诉带教老师哪里做错了              │
└────────────────────────────────────────┘
```

## 3. 情绪识别与需求表达卡模板

### 设计原则
- 使用表情图标直观展示情绪状态
- 使用颜色编码区分不同情绪（平静-绿色、紧张-黄色、焦虑-橙色、生气-红色）
- 需求表达使用简单的图示和短句
- 整体设计温馨、友好

### 模板示例
```
┌────────────────────────────────────────┐
│          情绪与需求表达卡              │
├────────────────────────────────────────┤
│                                        │
│  😊 平静：心情好，工作顺利             │

│                                        │
│  😰 紧张：心跳加快，手心出汗            │
│     停下来，深呼吸3次                   │
│     喝口水，休息1分钟                   │
│                                        │
│  😟 焦虑：担心做不好，害怕被批评         │
│     找到带教老师                        │
│     告诉老师：我有点担心                │
│                                        │
│  😠 生气：心里难受，想发脾气             │
│     离开现场，到休息区                  │
│     坐5分钟，慢慢平静                   │
│                                        │

│  👋 我需要帮助：老师，我不会做这个       │

```

## 4. 岗位培训手册模板

### 设计原则
- 使用大标题和清晰的层次结构
- 每个模块配有相关图标
- 使用简洁的语言和短句
- 重点内容加粗显示
- 整体布局整齐，便于阅读

### 模板示例
```
┌────────────────────────────────────────┐
│          岗位培训手册                  │
│        岗位：酒店保洁                   │
├────────────────────────────────────────┤
│                                        │
│  🏢 岗位认知：                         │
│  - 酒店保洁的工作是保持客房干净整洁     │
│  - 要对顾客友好，保持微笑              │
│                                        │
│  📋 工作流程：                         │
│  1. 敲门进入客房                        │
│  2. 整理床铺，换床单被罩                │
│  3. 清洁卫生间，擦镜子                  │
│  4. 吸尘，拖地                          │
│  5. 检查物品，补充消耗品                │
│                                        │
│  💬 沟通规范：                         │
│  - 遇到顾客说：您好！                   │
│  - 顾客问话要认真回答                  │

│                                        │
│  ⚠️ 安全规则：                         │

│  😊 情绪管理：                         │
│  - 工作累了可以休息一下                │

└────────────────────────────────────────┘
```

## 使用方法
1. 将生成的内容复制到模板中
2. 打印出来，制作成口袋卡或手册
3. 在工作现场随时参考使用
4. 根据实际情况进行适当调整

## 设计理念
- 符合心智障碍者的认知特点，使用简单、直观的视觉元素
- 减少文字量，增加图示和图标
- 使用颜色编码和表情符号增强理解
- 保持页面简洁，避免信息过载
- 内容可直接在工作现场使用，无需二次加工
