# 心智障碍者职场适配支持助手

## 项目简介
基于TRAE SOLO打造的零代码、零安装、现场即用、适配心智障碍者认知特点的轻量化职场支持工具。

## 核心功能模块
1. 岗位工作流程可视化拆解工具
2. 职场沟通辅助与话术生成工具
3. 情绪与需求表达支持工具
4. 岗位适配训练与岗前培训工具

## 支持的高频就业岗位
- 超市理货员
- 酒店保洁
- 烘焙店帮工
- 仓库分拣员
- 包装工
- 餐厅洗碗工
- 前台帮工
- 绿植养护
- 快递分拣
- 图书馆整理
- 文员助理
- 洗车工

## 使用说明
1. 打开TRAE SOLO
2. 复制系统提示词到SOLO中
3. 按照功能模块的专项Prompt进行操作
4. 生成的内容可直接打印或保存为图片使用

## 项目文件结构
- `/workspace/prompts/` - 系统提示词和专项Prompt
- `/workspace/examples/` - 功能示例
- `/workspace/docs/` - 项目文档
- `/workspace/test/` - 测试用例和验证报告
