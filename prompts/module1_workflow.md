# 岗位工作流程可视化拆解工具专项Prompt

## 功能说明
基于核心角色设定，为指定岗位的具体任务生成适配心智障碍者的操作指引，解决多步骤工作记不住、做不对的核心痛点。

## 专项Prompt
```
基于上述核心角色设定，为【岗位名称：{如超市理货员}】的【具体任务：{如临期商品下架}】，生成适配心智障碍者的操作指引，严格遵守以下规则：
1.  步骤拆分：将完整任务拆分为不超过8个核心步骤，单步骤只包含2个以内的动作，用短句表述，通俗易懂；
2.  固定输出结构：
- 任务核心目标（一句话说清要做什么、做到什么标准）
- 分步操作指引（编号+动作+完成标准，无复杂表述）
- 易错点提醒（不超过3条，每条一句话）
- 做错了怎么办（3步以内的应急处理方案，优先找带教老师）
3.  所有内容可直接打印成口袋卡，心智障碍者可照着步骤独立完成，无需二次修改。
```

## 使用示例
**用户输入：**
岗位名称：超市理货员
具体任务：临期商品下架

**预期输出：**
- 任务核心目标：把快过期的商品从货架上拿下来，交给带教老师
- 分步操作指引：
  1. 找到临期商品（看商品上的日期）
  2. 轻轻取下商品，放在手上
  3. 把商品拿到带教老师面前
  4. 告诉带教老师：这是临期商品
- 易错点提醒：
  1. 别拿错保质期好的商品
  2. 取商品时别弄掉其他商品
  3. 记得找带教老师，别自己处理
- 做错了怎么办：
  1. 立刻停下手中的事
  2. 找到带教老师
  3. 告诉带教老师哪里做错了
