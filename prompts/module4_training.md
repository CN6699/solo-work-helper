# 岗位适配训练与岗前培训工具专项Prompt

## 功能说明
基于核心角色设定，为指定岗位生成适配心智障碍者的岗前培训手册和训练题库，解决就业辅导资源紧缺、培训材料制作效率低的痛点，降低企业用工门槛。

## 专项Prompt
```
基于上述核心角色设定，为【岗位名称：{如酒店保洁}】生成岗位适配训练与岗前培训内容，严格遵守以下规则：
1.  一键生成岗位完整培训手册：包含岗位认知、工作流程、沟通规范、安全规则、情绪管理五大模块，可直接使用，无需二次修改；
2.  岗位模拟训练题库：生成岗位核心技能的模拟训练题、实操考核标准，可直接用于岗前培训和日常考核；
3.  企业用工适配指引：为爱心企业生成「心智障碍员工用工管理指引」，包含带教老师工作规范、环境适配建议、激励方法、风险防控等内容；
4.  所有内容必须用短句、大白话，单句不超过15个字，适配心智障碍者的认知特点，可直接在工作现场使用。
```

## 使用示例
**用户输入：**
岗位名称：酒店保洁

**预期输出：**
- 岗位完整培训手册：
  1. 岗位认知：
     - 酒店保洁的工作是保持客房干净整洁
     - 要对顾客友好，保持微笑
  2. 工作流程：
     - 敲门进入客房
     - 整理床铺，换床单被罩
     - 清洁卫生间，擦镜子
     - 吸尘，拖地
     - 检查物品，补充消耗品
  3. 沟通规范：
     - 遇到顾客说：您好！
     - 顾客问话要认真回答
     - 有问题找带教老师
  4. 安全规则：
     - 小心使用清洁剂，别弄到眼睛里
     - 拖地后要放防滑牌
     - 发现危险要告诉带教老师
  5. 情绪管理：
     - 工作累了可以休息一下
     - 心情不好找带教老师聊聊
- 岗位模拟训练题库：
  1. 如何正确整理床铺？
     - 答案：先把旧床单撤下来，再铺新床单
  2. 遇到顾客询问房间设施，应该怎么做？
     - 答案：微笑回答，不知道的找带教老师
  3. 清洁卫生间时要注意什么？
     - 答案：用对清洁剂，注意安全
- 企业用工适配指引（给爱心企业）：
  1. 带教老师工作规范：
     - 耐心教导，多鼓励
     - 分解任务，一步一步教
  2. 环境适配建议：
     - 工作区域保持整洁有序
     - 物品摆放固定位置
  3. 激励方法：
     - 完成任务给予表扬
     - 定期评选优秀员工
  4. 风险防控：
     - 定期安全培训
     - 建立应急处理机制
