# 情绪与需求表达支持工具专项Prompt

## 功能说明
基于核心角色设定，为心智障碍者生成情绪识别与需求表达支持内容，解决无法准确识别、表达自身情绪和需求的痛点，减少情绪应激事件。

## 专项Prompt
```
基于上述核心角色设定，生成情绪与需求表达支持内容，严格遵守以下规则：
1.  情绪识别与分级指引：用可视化的方式，帮助心智障碍者识别自己的情绪状态（平静/紧张/焦虑/生气），并对应生成极简的情绪舒缓步骤（如深呼吸3次、找带教老师、到休息区坐5分钟）；
2.  需求表达模板：生成「我需要帮助」「我身体不舒服」「我想休息5分钟」等标准化需求表达模板，配简单图示，心智障碍者可直接使用，无需组织语言；
3.  为带教老师生成「情绪应激事件处理指引」，提前预判风险，给出标准化的应对方法，避免事态升级；
4.  所有内容必须用短句、大白话，单句不超过15个字，可直接在工作现场使用。
```

## 使用示例
**预期输出：**
- 情绪识别与分级指引：
  1. 平静：心情好，工作顺利
     - 继续保持，认真工作
  2. 紧张：心跳加快，手心出汗
     - 停下来，深呼吸3次
     - 喝口水，休息1分钟
  3. 焦虑：担心做不好，害怕被批评
     - 找到带教老师
     - 告诉老师：我有点担心
  4. 生气：心里难受，想发脾气
     - 离开现场，到休息区
     - 坐5分钟，慢慢平静
- 需求表达模板：
  1. 我需要帮助：老师，我不会做这个
  2. 我身体不舒服：老师，我肚子有点疼
  3. 我想休息5分钟：老师，我有点累
  4. 我渴了：老师，我想喝水
  5. 我饿了：老师，我想吃饭
- 情绪应激事件处理指引（给带教老师）：
  1. 保持冷静，不要大声说话
  2. 走到心智障碍者身边，轻声询问
  3. 用简单的话安抚：「没关系，我来帮你」
  4. 带他到安静的地方，慢慢沟通
  5. 了解原因，给予具体帮助
