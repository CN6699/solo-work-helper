# 职场沟通辅助与话术生成工具专项Prompt

## 功能说明
基于核心角色设定，为指定岗位生成适配心智障碍者的职场沟通话术，解决不会说、不敢说、说不清的痛点，降低人际沟通焦虑。

## 专项Prompt
```
基于上述核心角色设定，为【岗位名称：{如烘焙店前台帮工}】生成职场沟通话术，严格遵守以下规则：
1.  覆盖4个核心场景：顾客常规提问、同事工作对接、向带教老师求助、表达自身需求；
2.  每个场景生成不超过5条高频话术，单条话术不超过20个字，用短句、大白话，心智障碍者可直接背诵、直接说出口；
3.  额外生成3条突发场景应急话术（如被顾客投诉、做错事被批评），话术必须简单、无攻击性，优先引导找带教老师；
4.  所有话术必须贴合岗位真实场景，可直接在工作中使用。
```

## 使用示例
**用户输入：**
岗位名称：烘焙店前台帮工

**预期输出：**
- 顾客常规提问话术：
  1. 您好，需要什么帮助？
  2. 这个面包是今天做的，新鲜。
  3. 收款机在这里，麻烦扫码。
  4. 谢谢光临，欢迎再来！
  5. 请问需要打包吗？
- 同事工作对接话术：
  1. 这个订单我做完了，给你。
  2. 请问这个放在哪里？
  3. 我去洗手间，帮我看一下。
  4. 这个面包卖完了，告诉顾客。
  5. 下班时间到了，交接一下。
- 向带教老师求助话术：
  1. 老师，我不会做这个，能教我吗？
  2. 老师，这个机器怎么用？
  3. 老师，顾客问的问题我不会答。
  4. 老师，我身体有点不舒服。
  5. 老师，我需要休息一下。
- 表达自身需求话术：
  1. 我想喝水，去拿杯子。
  2. 我肚子饿了，要去吃饭。
  3. 我眼睛累了，想休息5分钟。
  4. 我手疼，需要戴手套。
  5. 我要上洗手间，马上回来。
- 突发场景应急话术：
  1. 对不起，我去找带教老师。
  2. 老师，这里有顾客投诉我。
  3. 老师，我做错了，能帮我吗？
